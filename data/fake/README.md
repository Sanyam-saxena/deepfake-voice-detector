# Deepfake / AI-Generated Voice Samples

Place your **AI-generated or deepfake voice** audio files in this directory.

## Supported Formats
- WAV (`.wav`)
- MP3 (`.mp3`)
- FLAC (`.flac`)
- OGG (`.ogg`)
- M4A (`.m4a`)

## Examples
```
data/fake/
├── tts_output_001.wav
├── ai_generated_002.wav
├── deepfake_voice_01.mp3
└── synthetic_speech.wav
```

## Tips
- Include various types of AI-generated voices
- Use different TTS engines for diversity
- Ensure audio quality matches real samples
- Any duration is acceptable (will be processed to 3 seconds)
- Aim for at least 100+ samples for training

## Where to Find AI-Generated Voice Samples
- Text-to-Speech (TTS) tools (Google TTS, Amazon Polly, etc.)
- Voice cloning services
- ASVspoof dataset (https://www.asvspoof.org/)
- WaveFake dataset
- Generate your own using open-source TTS models
- Deepfake detection challenge datasets

## Popular TTS Tools for Generating Samples
- **ElevenLabs**: High-quality voice cloning
- **Google Cloud TTS**: Multiple voices and languages
- **Amazon Polly**: Natural-sounding TTS
- **Microsoft Azure Speech**: Enterprise-grade TTS
- **Coqui TTS**: Open-source TTS
- **Tortoise TTS**: High-quality open-source model
